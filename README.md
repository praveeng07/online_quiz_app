# online_quiz_app

A simple Android quiz application developed using Android Studio and SQLite.

## Features

- Multiple-choice quiz questions.
- SQLite database for storing questions and answers.
- User-friendly interface.
- Score tracking.


## Technologies Used

- Android Studio
- Java
- SQLite
